Camino Update

Sunday, July 24, 2005 12:25  
While it's hard for most people to see and know Camino has come such a long way. We the team really believe that 0.9 is actually 1.0 material. On the other hand we also know that if we release 0.9 soon 1.0 will just be a polish version.

We have som very nice things ramped up for the 0.9 beta release. We are trying to get the Java Embedding Plugin in the builds so that 99% of our java issues are solved. I'm also in cooperation with Nick Kreeger to finally implement Pause/Resume or downloads, we have a test build working so I think we can expect it soon. And there are other things which I won't mention just to make sure there are still some surpises when you download the build.
